topology (unique - optional): 
  • nodesPerLink (optional): number maximum of nodes per link (2 by default)
debug (unique, optional)
  • debugFlag(optional): Enable/disable debug message 
  • errorFlag(optional): Enable/disable error message
  • trace_dir(optional): directory for the ouput log files 
protocol (list): general or by default parameter for the protocol 
  • type: rupported types: "bgp 4" "TrafficInspectionTool", "tcp_full" 
    ----------bgp 4----------
  • parameters(unique): 
    • rtlog(optional): enable/disable route table log (optional)
    • dbglog(optional): enable/disable debug log
    • tracelog(optional): enable/diable message log
    • fsmlog(optional): enable/diable finite state machine log
    • tie_breaking(optional): defalut tie braking value or not
    • default_local_pref(optional): value for the attribute
    • med(optional): always_compare_med to compare always the med attribute 
    • routes_compare_level(optional): level of this parameter (Level 5 by default)
    • default_port(optional): 179 by default 
  • policyList (unique - optional)
    • policy
      • permit (optional, true/false. true by default): permit or deny a announced
      • id : id to reference the policy
      • clause (list): if all of them is unsatisfied, the route is denied. If one of this is satisfied, the following clauses aren't checked
        • id (optional) : id to refenrece this clause
        • predicate (unique - optional): condition to satisfied the clause. If this doesn't exist, clause is satisfied
          • atomicPredicate (list-optional): a condition to satisfied predicate. If this doen't exist, predicate is satisfied. 
            • id (optional): id to reference the atomic predicate.
            • attribute: attribute to compare. Attributes supported: "nlri", "origin", "as_path", "next_hop", "med", "local_pref", "community", "originator_id", "cluster_list"
            • match_string: value of the attribute
        • action (unique - optional). Actions to do with the route. If this doesn't exist, the route is permited
          • permit (optional true by default): true/false, if true permit the route and execute the atomic actions. If false not permit the route and the atomic actions aren't executed
          • atomicAction (list - optional): a action to execute
            • id (optional): id to reference the atomic action.
            • attribute: Attribute to execute action. Attributes supported: "as_path", "med", "local_pref", "community", 
            • action: ("set", "prepend", "append", "strip"). Action with the attribute
            • values: values to actions. Values are separated with spaces.
